Full Width Layout Test

This page tests that the layout is truly full-width on all devices

Full Width Section

This section should span the full width of the viewport on both desktop and mobile.

On desktop, you should see this content extend to the edges (with some padding).

On mobile, this should also be full width without extra side margins.

Navigation Test

Desktop: The navigation menu should be visible in the header and dropdown menus should be hidden until you click them.

Mobile: The hamburger menu should be visible, and clicking it should toggle the navigation menu visibility.

Dropdowns: Menu items with dropdowns should only show their submenu when clicked, not be visible all the time.

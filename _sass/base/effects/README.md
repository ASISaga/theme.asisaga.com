# Visual Effects

This directory contains visual effects - animations, futuristic effects, and atmospheric layers.

## Files

- `_animations.scss` - Core animation library (sacred rhythm, consciousness flow, etc.)
- `_futuristic-effects.scss` - Advanced glassmorphism, glows, quantum gradients
- `_ambient-layer.scss` - Sentient ambient atmosphere effects

## Purpose

These files provide the visual effects and animations that create the theme's distinctive aesthetic.

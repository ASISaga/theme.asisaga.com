# Utilities & Mixins

This directory contains utility mixins, helper functions, and accessibility tools.

## Files

- `_mixins.scss` - Core Sass mixins and utility functions
- `_semantic-mixins.scss` - Semantic mixins replacing Bootstrap
- `_accessibility.scss` - Accessibility features and WCAG compliance
- `_common.scss` - Common utility styles

## Purpose

These files provide reusable utilities and tools for theme development.

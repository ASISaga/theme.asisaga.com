# Layout System

This directory contains the layout system - responsive grid, containers, and layout structures.

## Files

- `_layout-wrappers.scss` - Modern layout containers (viewport wrappers)
- `_responsive-system.scss` - Modern responsive framework with container queries
- `_responsive.scss` - Responsive utilities and breakpoints
- `_layout.scss` - Base layout structures

## Purpose

These files provide the layout framework and responsive behavior for the theme.

# Product Components

This directory contains components specifically for product pages and product showcases.

## Files

- `_product-applications.scss` - Product applications section
- `_product-benefits-list.scss` - Product benefits listing
- `_product-card.scss` - Individual product card
- `_product-code-example.scss` - Code example sections
- `_product-feature-grid.scss` - Feature grid layout
- `_product-layout.scss` - Product page layout
- `_product-page.scss` - Product page specific styles
- `_product-section-container.scss` - Product section containers
- `_product-visual.scss` - Product visual elements

## Purpose

These components support the product marketing and documentation pages throughout the ASI Saga ecosystem.

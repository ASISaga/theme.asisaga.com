# Section Components

This directory contains components for major page sections and content blocks.

## Files

- `_cta.scss` - Call-to-action sections
- `_feature-grid.scss` - Feature grid sections
- `_hero.scss` - Hero sections
- `_section-header.scss` - Section headers
- `_sections.scss` - Generic section styles
- `_testimonial.scss` - Testimonial sections
- `_timeline.scss` - Timeline sections
- `_interactive-module.scss` - Interactive content modules

## Purpose

These components define major content sections that structure pages and provide engaging content displays.

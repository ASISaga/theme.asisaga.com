# Core Components

This directory contains fundamental UI components used throughout the theme.

## Files

- `_cards.scss` - Basic card components
- `_footer.scss` - Site footer
- `_header.scss` - Site header
- `_navbar.scss` - Navigation bar
- `_back-to-top.scss` - Back to top button

## Purpose

These are the foundational UI components that appear across multiple pages and layouts.

# Specialized Components

This directory contains domain-specific and special-purpose components.

## Files

- `_base-layout.scss` - Base layout utilities
- `_chatroom.scss` - Chat room interface
- `_layout-styles.scss` - Layout styling utilities
- `_linkedin.scss` - LinkedIn integration components
- `_specialized.scss` - Other specialized components
- `_utilities.scss` - Component utilities

## Purpose

These are specialized components for specific features or integrations that don't fit into other categories.

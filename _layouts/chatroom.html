--- layout: default module_js: chatroom-app.js --- {% comment %} Chatroom Layout — convenience wrapper for chat-based subdomains =============================================================== Maps Jekyll front-matter → chatroom-app component attributes via Liquid. The component itself handles all viewport CSS (chatroom-body / chatroom-main) and HTML templates; this layout adds no required infrastructure. A subdomain can use this layout for clean front-matter configuration, or skip it entirely and drop directly into any other layout. Front-matter reference: component - Custom-element tag (default: chatroom-app). Set to your subclass, e.g. `component: boardroom-app`. module_js - JS module to load (default: chatroom-app.js from this layout). Set to your module, e.g. `module_js: boardroom-app.js`. title - Chatroom title shown in the header. participants - Agent count badge. placeholder - Textarea placeholder. max_length - Maximum input length (default: 1000). show_toolbar - Boolean: show formatting toolbar. show_connection_status - Boolean: render live-connection badge. mcp_apps - JSON array of MCP app descriptors. mcp_endpoint - Fallback HTTP endpoint for all MCP apps. chat_data - Key into _data/chatroom/.yml for pre-loaded messages. api_endpoint - Live API base URL. auto_refresh - Boolean: poll api_endpoint. refresh_interval - Polling interval ms (default: 3000). owner - Workflow owner shown in header. step_id - Current step identifier. total_steps - Total steps for progress indicator. theme - Applies chatroom--theme- CSS class. show_toggle_strip - Boolean: set show-toggle-strip attribute. show_members_sidebar - Boolean: set show-members-sidebar attribute. show_agent_profiles - Boolean: set show-agent-profiles attribute. enable_screen_share - Boolean: set enable-screen-share attribute. enable_video_call - Boolean: set enable-video-call attribute. enable_file_attach - Boolean: set enable-file-attach attribute. enable_formatting - Boolean: set enable-formatting attribute. boardroom.api_base - Boardroom REST API base URL. boardroom.copilotkit_runtime_url - CopilotKit AG-UI runtime URL.

It includes metadata, tags, and structured content perfect for blog entries and news articles. ## What is a Post Layout? The post layout is designed for blog posts and news entries. It provides: - Post metadata (date, author, reading time) - Tag and category support - Social sharing options - Navigation to previous/next posts ## Key Features ### 1. Metadata Display The layout automatically displays post metadata at the top of the page, including publication date, author information, and estimated reading time. ### 2. Tag Support Tags and categories are displayed as badges, making it easy for readers to find related content. ### 3. Responsive Design The layout is fully responsive and looks great on all devices. ## Conclusion This post layout provides everything you need for a professional blog setup with minimal configuration.
